## NaViTest-2021寒假考核-2020215163-田奥华

### App简要介绍

该NaViTest高德Demo实现了

1. 获取当前定位信息
2. 显示定位地图
3. 地图图层修改
   - 修改为卫星图层
   - 修改为夜间图层
   - 修改为白天图层
4. 获取POI数据
5. 绘制marker点
   - 清除marker点
6. 地图点击事件
   - 显示地址
7. 地图点击长按事件
   - 显示经纬度
8. 搜索目的地并显示该地经纬度
9. 导航功能
   - 步行路线规划
   - 步行时间和距离的Toast
